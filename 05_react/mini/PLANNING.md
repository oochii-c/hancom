# Visual Builder 기획안

## 프로젝트 개요
매번 레이아웃을 짜서 코드로 요청하는 과정이 번거로워, 캔버스 위에서 도형·텍스트를 그림판/PPT처럼
직접 배치·편집하고 그 결과를 HTML 또는 JSX 코드로 바로 내려받는 프론트엔드 도구.

## MVP 범위 (최초 기획)
- **왼쪽 툴바**: 박스 / 원 추가 버튼
- **캔버스**: 요소 드래그 이동, 모서리 핸들로 리사이즈, 클릭으로 선택
- **오른쪽 속성 패널**: 선택된 요소의 색상 / 크기 / 위치 숫자 입력
- **상단 "코드로 내보내기" 버튼**: HTML 또는 JSX 텍스트 생성 + 클립보드 복사

## 추가된 기능 (진행 순서대로)

### 1차 확장
1. **겹침 방지 배치** — 새 요소를 추가하면 기존 요소와 겹치지 않는 빈 자리에 자동 배치
2. **텍스트 요소** — 도형 외에 텍스트도 캔버스 요소로 추가 가능
3. **텍스트-도형 병합 팝업** — 텍스트를 도형 위로 드래그해 겹치면 "텍스트를 도형에 포함시킬까요?" 확인 팝업, 승인 시 텍스트가 도형 안 라벨로 흡수
4. **디스플레이 크기 조절** — Desktop(1920×1080) / Laptop / Tablet / Mobile 프리셋 + 커스텀 너비·높이 직접 입력

### 2차 확장
5. **병합 UI 배치 순서** — 텍스트가 도형에 합쳐지면 텍스트 관련 조작 UI가 도형 조작 UI *아래*에 노출
6. **내보내기 히스토리** — 이전에 내보낸(복사한) 코드를 히스토리 페이지에서 시간순으로 다시 확인·복사 가능
7. **외곽선** — 도형·텍스트에 외곽선 두께/색상 추가, 모서리 굴절도(border-radius) 조절 가능
8. **투명도(Opacity)** — 색상에 opacity 슬라이더 추가

### 3차 확장
9. **합치기/분리 폴딩 UI** — 텍스트-도형이 합쳐지면 속성 패널의 "포함된 텍스트" 영역이 폴딩(접기/펼치기) 형태로 전환되고, 폴딩 바 끝에 "분리" 버튼 배치
10. **텍스트 효과** — 하이라이트, 볼드, 이탤릭, 헤딩(H1~H4) 적용 가능 (독립 텍스트 요소·도형에 병합된 텍스트 모두 지원)

## 기술 스택
- React (Create React App)
- 순수 CSS (외부 UI 라이브러리 없음)
- 상태 관리: React 기본 useState/useRef (별도 라이브러리 없음)
